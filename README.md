# 🎵 Media Player App

## 📌 Description
This is an Android application developed as part of the Mobile Application Development assignment.  
The app allows users to:
- Play audio files from device storage
- Stream video from a URL
- Control playback using Play, Pause, Stop, and Restart buttons

---

## 🚀 Features
- 🎵 Audio playback from device
- 🎬 Video streaming from URL
- ⏯ Playback controls:
  - Play
  - Pause
  - Stop
  - Restart
- 📱 Simple and user-friendly interface

---

## 🛠️ Technologies Used
- Java
- Android Studio
- MediaPlayer API
- VideoView

---

## 📂 Project Structure
- MainActivity.java → Handles audio & video logic
- activity_main.xml → UI design
- AndroidManifest.xml → Permissions & app configuration
- strings.xml → Text resources
